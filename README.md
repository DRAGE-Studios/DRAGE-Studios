# Welcome to DRAGE Studios

DRAGE Studios is a small team based out of Melbourne Australia. We work on projects big and small for people to use and enjoy.

## Projects
#### DRAGE Anti-Cheat (DAC) [05.05.2025 - 22.02.2026]
- DRAGE Anti-Cheat, or DAC for short, was a Roblox and third-party database anticheat system made for games made by us. The anticheat would use third-party databases to store ban data and had individual modules for easy integration with any game of ours. It ment that people would be banned from all games if they were banned from one.
#### Syndicate Suite [22.02.2026 - Present]
- Syndicate Suite is an online web application made for FiveM criminal organization and syndicate roleplays. It allows criminal organizations (within FiveM roleplay servers) to be able to manage tasks, finances, inventory and users all in one dashboard along with automations for webhooks sent to Discord, and API keys for information access and editing of the organization.
#### Live News [2023 - 2024]
- Live News was an online web platform that was dedicated to providing more local based news. This application was supposed to be powered by local students, but never finished development.
#### **** Resources [2024-2025]
- Retracted name for this project. Basically was an application to ease file sharing between people. Finished development, closed.
